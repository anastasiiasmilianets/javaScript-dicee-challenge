# Dicee Game 🎲

A simple interactive dice game web application built to practice DOM manipulation and event handling using native JavaScript. Click the button to roll the dice for two players, see the winner announced instantly, and track the running score across multiple rounds.

---

## 🚀 Features
* Interactive Rolling: Click the "Roll the dice" button to generate a random number (1–6) for each player — no page refresh needed.
* Dynamic DOM Updating: The dice images and the main heading (<h1>) update automatically based on the outcome.
* Score Tracking: Wins for each player are counted and displayed live, persisting across rolls until the page is reloaded.
* Three End-Game States: Displays a win for Player 1, a win for Player 2, or a Draw.

---

## 🛠️ Tech Stack
* **HTML5** — page structure.
* **CSS3** — custom styling, layout, and Google Fonts (Roboto, Unna).
* **Vanilla JavaScript** — event handling (addEventListener), random number logic, and DOM manipulation using querySelector.

---

## 📂 Project Structure
* `dicee.html` — the main game page.
* `styles.css` — styling and layout definitions.
* `index.js` — JavaScript file containing the game logic (roll handler and score tracking).
* `images/` — folder containing the dice images (from `dice1.png` to `dice6.png`).
