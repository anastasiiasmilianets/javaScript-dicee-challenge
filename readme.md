# Dicee Game 🎲

A simple dice game web application built to practice DOM manipulation using native JavaScript. Every time the page is refreshed, a random dice value is generated for two players, and the winner is automatically determined.

---

## 🚀 Features
* **Random Generation:** Each die rolls a random number between 1 and 6 upon page refresh.
* **Dynamic DOM Updating:** The game automatically changes the dice images and the main heading (`<h1>`) based on the outcome.
* **Three End-Game States:** Displays a win for Player 1, a win for Player 2, or a Draw.

---

## 🛠️ Tech Stack
* **HTML5** — page structure.
* **CSS3** — responsive styling and custom Google Fonts (Roboto, Unna).
* **Vanilla JavaScript** — random number logic and DOM manipulation using `querySelector`.

---

## 📂 Project Structure
* `dicee.html` — the main game page.
* `styles.css` — styling and layout definitions.
* `index.js` — JavaScript file containing the game logic.
* `images/` — folder containing the dice images (from `dice1.png` to `dice6.png`).
